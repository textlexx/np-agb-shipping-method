ТЕХНІЧНІ ВИМОГИ ДЛЯ ЗАПУСКУ ПЛАГІНУ:

1. PHP 8.4 (версія повинна бути точно більша ніж 7.4, і бажано 8.4 бо на 8.0-8.3 не тестувалось)

2. WORDPRESS 7

3. WOOCOMMERCE 10.8

----------------------------------

ВСТАВНОЛЕННЯ:

1. Завантажити проект з гіт.

2. Папку з проектом розмістити в папку WORDPRESS: /wp-content/plugins.

3. В панелі керування WORDPRESS в розділі плагінів знайти плагін - Nova Poshta AGB Shipping Method

4. Активувати плагін. 

5. В налаштуваннях WOOCOMMERCE -> перейти в розділ "доставка" -> створити зону доставки -> вибрати "додати метод доставки" -> вибрати метод доставки "нова пошта" -> налаштувати сам метод -> зберегти -> користуватись.

----------------------------------

Розроблено тільки плагін Nova Poshta AGB Shipping Method, WORDPRESS і інші плагіни ніяким чином не змінювались.
